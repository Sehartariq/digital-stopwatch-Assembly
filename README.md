# Digital Stopwatch in x86 Assembly

A digital stopwatch application written in **x86 Assembly language** for **EMU8086**.

## 🎯 Features
- Start/Stop/Pause/Reset functionality
- Real-time display (MM:SS format)
- Non-blocking keyboard input
- BIOS interrupt-based timing

## 🎮 Controls
| Key | Action |
|-----|--------|
| `S` | Start timer |
| `P` | Pause timer |
| `R` | Reset to 00:00 |
| `Q` | Quit program |

## 🛠️ How to Run
1. Download and install [EMU8086](https://emu8086-microprocessor-emulator.en.softonic.com/)
2. Open `stopwatch.asm` in EMU8086
3. Click **Emulate** or press `F5`
4. Use keyboard controls (S, P, R, Q)
